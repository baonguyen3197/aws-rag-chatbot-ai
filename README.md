=====================================
## Setting Up the Project Environment
=====================================
```bash
pyhon -m venv venv
source venv/bin/activate  # On Windows use `venv\Scripts\activate`
pip install -r requirements.txt
```

=====================================
## Initializing Reflex Project
=====================================
```bash
reflex init
```

=====================================
## Run Reflex App
=====================================
```bash
reflex run
```
